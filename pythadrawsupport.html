Welcome to the Pytha Draw Support Page

Pytha Draw is an educational mobile game designed to help you build mathematical intuition by drawing functions directly on a graph.

By playing, you progressively master polynomials (degree 1, 2, and 3), along with related concepts such as:

  • derivatives
  • canonical forms
  • roots
  • variations
  • limits
  • comparative growth

If you need guidance or encounter an issue, you are in the right place.

Playing Modes

Pytha Draw includes several game modes, each training a specific mathematical skill.

Classic Mode

Draw the function directly without visual aids. Focus on precision or speed, and discover techniques step by step in Learn mode.

Best for: learning curve-reading and improving drawing accuracy.

Derivative Mode

Draw the function using only its derivative. Each level reveals essential information about how the original function behaves.

Best for: understanding variations and extrema.

Canonical Mode

Transform the equation into a canonical form where the graph becomes directly readable. This makes the curve easier to interpret and draw accurately.

Best for: factorisation, structure recognition, rapid computations.

Addition Mode

Break the expression apart to draw more accurately. Mentally add the missing right part of the equation to reconstruct the perfect curve.

Best for: decomposition, mental manipulation, and precise plotting.

Guess Mode

The game is reversed. Reconstruct the equation from the drawing, only one attempt to get it right.

Best for: linking visual shape to algebraic form.

Learn Mode and Tutorials

Learn Mode is the guided learning path in Pytha Draw. It introduces concepts progressively through short interactive steps, combining:

  • clear explanations (what to look for on the graph),
  • guided practice (draw with purpose, not by guessing),
  • quick checks (mini-questions when needed),
  • and immediate feedback (so you learn from each attempt).

What Learn Mode helps you improve

  • reading and drawing polynomial curves (degree 1, 2, 3),
  • connecting algebra â†” graph,
  • recognizing key features (slope, roots, vertex, extrema),
  • and applying techniques used in the other game modes.

How to access Learn Mode

  1. Open the app and go to Learn.
  2. Select a tutorial category (Linear, Quadratic, etc.).
  3. Follow the steps in order.

Tips if a tutorial feels difficult

  • Replay the same tutorial a second time: the goal is reflex, not memorization.
  • Focus on one key feature per attempt (roots, vertex, slope, extrema).
  • Use Learn Mode first, then return to Classic/Derivative/Canonical modes to reinforce.

Progression tracking

As you complete levels, you unlock radar visualizations that help you:

  • measure your progress over time
  • identify strengths and weaknesses
  • target the concepts you want to improve

Your score typically depends on:

  • precision (how close you are to the drawn solution)
  • speed (how long do you take to draw)

You can also track your performance through:

  • global leaderboards
  • private leagues with friends or classmates

Data and Account Management

Pytha Draw does not collect personal data. You only choose a username (pseudo) used for:

  • ranking
  • game progression
  • league participation

Delete your data

You can delete all your data directly from the in-app settings. This permanently removes your:

  • username
  • scores
  • radars
  • progression
  • league participation

No email request is necessary. This action is permanent and cannot be undone.
